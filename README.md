## Seminar overview
 
Published DL-based docking tools are confounded by differences in training data, making fair comparison difficult​.
In this seminar, each student will become an expert and present a tool, retrain and evaluate an assigned docking tool on a common dataset under standardized conditions.​

The seminar consists of three sessions and two submission deadlines, held on **Fridays** at **10:00 am** in CBI (E2.1), room 001.
